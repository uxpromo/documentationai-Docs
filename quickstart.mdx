## Prerequisites

Before starting, ensure you have:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account creation
- Stable internet connection

</Callout>

No additional software installation is required for initial access.

## Create Your Finzor Account

Follow these steps to sign up on the `ui2.finzor.dev` subdomain.

<Steps>
  <Step title="Visit the Site" icon="globe">
    Open your browser and navigate to [ui2.finzor.dev](https://ui2.finzor.dev).
  </Step>
  <Step title="Start Signup" icon="user-plus">
    Click the `Sign Up` or `Create Account` button on the landing page.
  </Step>
  <Step title="Enter Details" icon="edit-3">
    Provide your email, create a strong password, and complete any CAPTCHA if prompted.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Finzor and click the link to activate your account.
  </Step>
</Steps>

You now have a Finzor account ready for login.

## Log In and Access the Dashboard

<Steps>
  <Step title="Return to Login" icon="log-in">
    Go back to [ui2.finzor.dev](https://ui2.finzor.dev) and click `Log In`.
  </Step>
  <Step title="Enter Credentials" icon="key">
    Input your email and password, then submit.
  </Step>
  <Step title="View Default Page" icon="layout">
    You land on the default dashboard. This early-stage interface shows core navigation elements.
  </Step>
</Steps>

<Callout kind="tip">

  Bookmark `ui2.finzor.dev` for quick access. Enable two-factor authentication if available in settings.

</Callout>

## Navigate the Early UI

The current UI is minimal but functional. Key areas include:

- **Sidebar Menu**: Access account settings and logout.
- **Main Content**: Displays welcome message and basic stats.
- **Header Bar**: User profile and notifications.

Use your browser's developer tools for inspection:

<Tabs>
  <Tab title="Chrome/Edge" icon="chrome">
    Right-click any element and select `Inspect`. Open the Console tab to run JavaScript.
  </Tab>
  <Tab title="Firefox" icon="firefox">
    Right-click and choose `Inspect Element`. Switch to Console for scripting.
  </Tab>
</Tabs>

## Basic Development Setup

For deeper interaction, test API connectivity from the browser console.

<CodeGroup tabs="Browser Console,Node.js">
  ```javascript
  // Fetch user profile from Finzor API
  fetch('https://api.finzor.dev/v1/user/profile', {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  })
    .then(response => response.json())
    .then(data => console.log('Profile:', data));
  ```
  ```javascript
  // Node.js example with fetch
  const fetch = require('node-fetch');

  async function getProfile() {
    const response = await fetch('https://api.finzor.dev/v1/user/profile', {
      headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
    });
    const data = await response.json();
    console.log('Profile:', data);
  }

  getProfile();
  ```
</CodeGroup>

Replace `YOUR_TOKEN` with your actual auth token from the dashboard.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained after login.
</ParamField>
