## Quick Start

Get up and running with Finzor in minutes. Follow these steps to explore the platform.

<Steps>
  <Step title="Visit the UI" icon="globe">
    Open `ui2.finzor.dev` in your browser. You see the welcome page confirming active development.
  </Step>
  <Step title="Sign Up" icon="user-plus">
    Create an account using your email. Verify via the confirmation link sent to you.
  </Step>
  <Step title="Generate API Key" icon="key">
    Head to your dashboard settings. Click "Generate Key" to obtain `YOUR_API_KEY`.
  </Step>
  <Step title="Test Integration" icon="play">
    Use the example below to make your first API call.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/status', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const data = await response.json();
  console.log(data);
  ```
  ```bash
  curl -X GET https://api.example.com/v1/status \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```
</CodeGroup>
