## Recent Updates

Finzor evolves rapidly in its early stages. Review the latest releases below to understand new capabilities, fixes, and changes. You receive automatic notifications for major updates via the dashboard at `https://dashboard.finzor.dev`.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>
## New Features

- Introduce real-time dashboard widgets for monitoring API usage and performance metrics.
- Add webhook support for event notifications, including user sign-ups and payment events.
- Enable custom themes with your brand color `#3B82F6`.

## Improvements

- Optimize load times by `>30%` through lazy loading of UI components.
- Enhance search functionality across logs and analytics with fuzzy matching.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>
## New Features

- Launch API v2 endpoints at `https://api.finzor.dev/v2` with improved rate limiting.
- Integrate one-click OAuth flows for Google and GitHub authentication.

## Bug Fixes

- Resolve intermittent failures in session persistence during high traffic.
- Fix validation errors when submitting forms with optional fields.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>
## Initial Release

- Core dashboard with analytics, user management, and settings.
- Basic API for data ingestion at `https://api.finzor.dev/v1`.
- Secure authentication using JWT tokens.

## Breaking Changes

- Default API version changed from beta to v1.

</Update>

## Migration Guide

Upgrade seamlessly between versions. Follow these steps to apply updates without downtime.

<Callout kind="alert">
Backup your configuration and database before migrating. Test in staging first.
</Callout>

<Steps>
  <Step title="Check Compatibility" icon="check-circle">
    Review release notes for breaking changes.

    <CodeGroup tabs="JavaScript,Python">
    ````javascript
    // Verify API version
    const response = await fetch('https://api.finzor.dev/v2/health');
    console.log(response.status);
    ````
    ````python
    import requests
    response = requests.get('https://api.finzor.dev/v2/health')
    print(response.status_code)
    ````
    </CodeGroup>
  </Step>

  <Step title="Update SDK" icon="download">
    Install the latest client library.
  </Step>

  <Step title="Deploy Changes" icon="rocket">
    Restart services and monitor logs.
  </Step>
</Steps>

<Expandable title="Advanced Migration for v1.2.0" default-open="false">
If using webhooks, update your endpoint URL to include the new signature header:

````javascript
// Add signature verification
const signature = req.headers['x-finzor-signature'];

````
</Expandable>
